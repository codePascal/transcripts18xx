Welcome to transcripts18xx documentation
========================================

A Python package to parse and process game transcripts
from `18xx.games <https://18xx.games/>`_.
It produces structured data suitable for analysis, modeling and visualization.

The following games are supported:

* `1830: Railroads & Robber Barons <https://en.wikipedia.org/wiki/1830:_The_Game_of_Railroads_and_Robber_Barons>`_
